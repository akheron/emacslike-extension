# emacs-like-utils

Helpers for those who are into Emacs.

## Features

* Find file in the spirit of `counsel-find-file`: `emacslike.findfile.show`
* Search in fles in the spirit of `ag.el`: `emacslike.ag.show`

## Requirements

`ag` must be installed to search in files.
